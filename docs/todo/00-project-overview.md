## 비전

> "운명을 금융 데이터처럼 분석하는 새로운 경험"

사주라는 동양의 전통적 빅데이터를 토스라는 현대적 금융 플랫폼의 문법으로 번역하여, MZ세대가 자신의 운명을 '믿는' 것이 아니라 '소비'하고 '분석'하는 서비스.

---

## 핵심 차별화 포인트

### 1. 하이브리드 프로파일링 (Saju x MBTI)
- 사주의 십성(Ten Gods)을 MBTI 4가지 지표와 매핑
- **운명-현실 괴리 분석 (Gap Analysis)**: 선천적 기질(사주) vs 현재 성향(MBTI) 비교
- "숨겨진 잠재력" 인사이트 제공

### 2. 데이터 시각화 중심 UX
- 기존 앱: "역마살이 있어 이동수가 많습니다"
- Destiny.OS: "2026년 이동성 지수가 전년 대비 40% 상승했습니다"
- 월별 운세를 주가 차트처럼 시각화

### 3. 토스 디자인 시스템 준수
- Pretendard 폰트
- 오방색의 파스텔 톤 재해석
- Tossface 이모지 활용

### 4. AI 상담 기능
- 1회 무료 AI 프롬프트 상담 제공
- 개인화된 운세 해석 및 조언

## 반드시 먼저 확정해야 하는 4가지 (리스크 탑)

1) **토스 미니앱에서 Flutter가 가능한지**(가능하지 않으면 “미니앱=웹” 등으로 스택이 바뀝니다)
2) **AI(OpenAI 등) 호출 위치**: 앱 직접 호출 금지(키 유출/남용) → **서버리스 함수 경유** 권장
3) **생년월일시 저장 정책**: 기본값은 **로컬 Secure Storage**, 서버 저장은 선택/명시적 동의
4) **60갑자/만세력 계산의 기준 표준**: 간단 공식으로 때우면 오차/클레임이 생깁니다(표준 테이블/검증 필요)
