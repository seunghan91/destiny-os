# Destiny.OS - UI/UX 설계

## 디자인 원칙

### 토스 디자인 시스템 (TDS) 준수

> "급진적인 단순함 (Radical Simplicity)"

1. **데이터 시각화 중심**: 텍스트 나열 → 차트/그래프
2. **미니멀 UI**: 한 화면에 하나의 목적
3. **친근한 톤**: 신비주의 탈피, 현대적 표현
4. **일관된 인터랙션**: 토스 앱과 동일한 제스처/피드백

## 화면 플로우

```
Splash → Onboard (첫 방문) → Input → Result Hub
                                      ├── 사주 요약 → 상세 사주
                                      ├── MBTI 분석 → Gap 분석
                                      ├── 2026 운세 → 월별 차트
                                      ├── 대운 타임라인 → 대운 상세
                                      ├── 궁합 분석
                                      ├── AI 상담
                                      └── 공유 카드
```

## 주요 화면 설계

### 1. 입력 화면

**구성 요소**:
- 생년월일: 커스텀 휠 피커 + 햅틱 피드백
- 시간: 12시진 기반 선택지 (선택)
- 성별: 토글 버튼
- MBTI: 드롭다운 또는 16개 그리드 (선택)

**인터랙션**:
- 휠 스크롤 시 미세한 진동 (Tick)
- 선택 완료 시 중간 진동 (Medium Impact)

### 2. 결과 허브 화면

**레이아웃**:
- 상단: 2026년 총운 메인 카드
- 중단: 2x2 그리드 네비게이션
  - 사주 요약 | MBTI 분석
  - 대운 타임라인 | 궁합 분석
- 하단: AI 상담 버튼

### 3. 2026년 운세 상세

**구성 요소**:
- 병오년(火馬) 아이콘 + 제목
- 월별 에너지 차트 (fl_chart)
- 최고점/최저점 어노테이션
- 분야별 점수 바 (직업/재물/연애/건강)

### 4. MBTI Gap 분석

**구성 요소**:
- 사주 기반 MBTI vs 실제 MBTI 비교 카드
- 괴리 점수 프로그레스 바
- 인사이트 텍스트 카드
- 2026년 액션 아이템

### 5. 대운 타임라인

**구성 요소**:
- 가로 스크롤 타임라인 (10년 단위)
- 현재 대운 하이라이트 (파란색 마커)
- 터치 시 상세 팝업 카드
- 다음 대운까지 카운트다운

### 6. 공유 카드

**규격**:
- 9:16 비율 (인스타 스토리 최적화)
- RepaintBoundary로 위젯 → 이미지 변환
- 토스 디자인 시스템 적용
- 앱 브랜딩 + QR코드/링크 포함

## 컴포넌트 라이브러리

### 핵심 위젯

| 위젯명 | 용도 |
|--------|------|
| DestinyCard | 기본 카드 컴포넌트 |
| EnergyChart | 월별 에너지 차트 |
| DateWheelPicker | 날짜 선택 휠 |
| PillarView | 사주 기둥 표시 |
| ScoreBar | 점수 막대 그래프 |
| TimelineView | 대운 타임라인 |

### 마이크로 인터랙션

| 상황 | 피드백 |
|------|--------|
| 휠 스크롤 | HapticFeedback.selectionClick() |
| 선택 완료 | HapticFeedback.mediumImpact() |
| 결과 공개 | HapticFeedback.heavyImpact() + 카드 플립 |
| 숫자 표시 | 카운트업 애니메이션 |
| 차트 표시 | 드로잉 애니메이션 |
