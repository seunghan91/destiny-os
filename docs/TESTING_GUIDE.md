## 🚨 문제 발생 시

### 문제 1: 웹앱 로드 안 됨

```
해결:
1. 브라우저 새로고침 (Ctrl+Shift+R / Cmd+Shift+R)
2. Firebase Console 확인:
   https://console.firebase.google.com/project/destiny-os-2026
3. 배포 상태 확인
```

### 문제 2: 로그인 불가

```
확인:
1. Firebase 프로젝트 확인
2. OAuth 클라이언트 ID 설정 확인
3. 웹앱 승인된 도메인 확인
```

### 문제 3: 데이터 저장 안 됨

```
확인:
1. Supabase API 연결 확인 (.env)
2. RLS 정책 확인
3. 네트워크 탭에서 API 요청 확인
```

### 문제 4: Admin 페이지 비밀번호

```
기본값: destiny2026
변경: admin_page.dart에서 _adminPassword 수정
```
